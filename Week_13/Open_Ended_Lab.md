# Open Ended Android App Development Lab

## Objective

Students will design, develop, and deploy a fully functional Mobile
application of their own choice using modern Mobile development
practices.

## Problem Statement

Identify a real-world problem and design an Android mobile application
that provides a meaningful solution.

## Minimum Requirements

-   Minimum 3 Activities / Screens\
-   Navigation using Intents or Navigation Component\
-   Data Storage using SQLite / Room / Firebase / SharedPreferences\
-   List display using RecyclerView or LazyColumn\
-   User interaction using forms, buttons, dialogs\
-   At least one advanced feature: API, Camera, Location, Sensors,
    Notifications, Firebase Auth

## Technology Stack

-   Language: Java or Kotlin or Flutter
-   UI: XML or Jetpack Compose\
-   Architecture: Free choice

## Deliverables

-   APK File\
-   Source Code\
-   Project Report (PDF)\
-   Presentation Slides\
-   Live Demo

## Evaluation Rubric (50 Marks)

  Criteria              Marks
  --------------------- -------
  Innovation & Idea     10
  UI/UX Design          10
  Functionality         10
  Android Features      10
  Presentation & Demo   10

## Sample App Ideas

-   Smart Attendance System\
-   Expense Tracker\
-   Campus Event Manager\
-   Health Monitoring App\
-   Emergency Alert App

## Viva Questions

1.  Why did you choose this problem?\
2.  Explain your app architecture.\
3.  How does your database work?\
4.  What challenges did you face?\
5.  What future improvements are planned?
